# Claude Code トレーニングリポジトリ

## 目的
Claude Code の活用スキルを体系的に学習・実践するためのリポジトリ。
学習教材、評価システム、リサーチ資料を一元管理する。

## リポジトリ構造

### 学習教材（ルートの .md ファイル群）
- `00_全体マップ.md` - カリキュラム全体の構成
- `01_基礎と設定.md` 〜 `09_コストと運用.md` - 各ドメインの教材

### eval_system/ - 活用度評価システム
- `scoring_model.md` - 評価基準（1000点満点、9ドメイン）
- `eval_data.md` - 全プロジェクトから収集した設定データ
- `eval_report.md` - 評価結果レポート
- `improvements/` - プロジェクト別改善計画
- `agents/` - 評価用マルチエージェント構成

### reference/ - 参考資料
### research/ - 技術調査メモ

## 作業ルール
- 教材の内容を変更する場合は変更理由を明記する
- eval_system/ のスコアリングは scoring_model.md の基準に厳密に従う
- eval_data.md には実データのみ記載する（推測・補完をしない）
- 改善計画にはコピペ可能なコマンド・設定を含める
