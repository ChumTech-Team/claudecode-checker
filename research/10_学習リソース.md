## 2. Claude Certified Architect（CCA）認定試験

### 概要
- **正式名称**: Claude Certified Architect -- Foundations（CCA-F）
- **ローンチ**: 2026年3月
- Claude モデルファミリーで本番グレードアプリケーションを構築するエンジニア向けの**初の専門認定**
- **受験料**: **$99/回**

### 試験形式
- **問題数**: 60問（多肢選択式、各問に正解1つ・ディストラクター3つ）
- **制限時間**: 120分（2時間）
- **形式**: プロクター付き、クローズドブック、AI支援不可
- **スコア範囲**: 100 - 1,000
- **合格スコア**: **720**
- **シナリオ**: 6つのシナリオのうち**4つがランダムに選択**され、全問がそのシナリオに紐づく

### 5ドメインと配点

| ドメイン | 内容 | 配点 |
|---|---|---|
| **Domain 1: Agentic Architecture & Orchestration** | マルチエージェントシステム設計、タスク分解、ハブ&スポークモデル | **27%** |
| **Domain 2: Tool Design & MCP Integration** | MCPサーバー設計、ツール境界管理、推論過負荷防止 | **18%** |
| **Domain 3: Claude Code Configuration & Workflows** | Claude Code設定、開発ワークフロー | **20%** |
| **Domain 4: Prompt Engineering & Structured Output** | プロンプトエンジニアリング、構造化出力 | **20%** |
| **Domain 5: Context Management & Reliability** | コンテキスト管理、信頼性 | **15%** |

> **上位3ドメインでスコアの67%を占める**

### 6つの試験シナリオ
1. Customer Support Resolution Agent
2. Code Generation with Claude Code
3. Multi-Agent Research System
4. Developer Productivity with Claude
5. Claude Code for CI/CD
6. Structured Data Extraction

### 早期アクセス
- Claude Partner Network メンバーの最初の **5,000名の従業員**は無料で受験可能
- パートナーネットワークへの参加は無料

**ソース**: https://www.lowcode.agency/blog/how-to-become-claude-certified-architect, https://claudecertifications.com/
