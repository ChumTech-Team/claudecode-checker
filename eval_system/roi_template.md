### ROI計算式

```
月間節約時間（時間） = SUM(各タスクの月間節約時間)
時間単価（円/時間） = {時給 or 想定単価}
月間節約金額 = 月間節約時間 x 時間単価

月間コスト = サブスク + API従量 + ツール費
月間ROI = (月間節約金額 - 月間コスト) / 月間コスト x 100

例:
  月間節約: 40時間 x ¥5,000 = ¥200,000
  月間コスト: ¥30,000（Max プラン $200 相当）
  ROI = (200,000 - 30,000) / 30,000 x 100 = 567%
```
