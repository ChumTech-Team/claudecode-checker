# メール下書きテンプレート

> Claude Code でメール下書きを依頼する際に使用するテンプレート集

---

## 使い方

Claude Code に以下のようにプロンプトを投げる:

```
以下のテンプレートに沿ってメールを作成してください。
- 種別: [依頼/報告/お礼/謝罪/案内]
- 宛先: [相手の名前・役職]
- 件名: [概要]
- 本文の要点: [箇条書きで]
- トーン: [丁寧/カジュアル/フォーマル]
```

---

## テンプレート1: 依頼メール

```
件名: 【依頼】{件名}

{宛先の名前} 様

お疲れ様です。{自分の名前}です。

{背景・経緯を1-2文}

つきましては、以下の件についてご対応いただけますでしょうか。

■ 依頼内容
- {依頼事項1}
- {依頼事項2}

■ 期限
- {期限}

■ 備考
- {補足情報}

お忙しいところ恐縮ですが、ご確認のほどよろしくお願いいたします。

{自分の名前}
```

---

## テンプレート2: 進捗報告メール

```
件名: 【報告】{プロジェクト名} 進捗報告（{日付}）

{宛先の名前} 様

お疲れ様です。{自分の名前}です。
{プロジェクト名}の進捗をご報告いたします。

■ 全体進捗: {XX}%

■ 今週の成果
- {成果1}
- {成果2}

■ 課題・リスク
- {課題1}（対応策: {対応}）

■ 来週の予定
- {予定1}
- {予定2}

ご不明点がございましたら、お気軽にお声がけください。

{自分の名前}
```

---

## テンプレート3: お礼メール

```
件名: {件名}のお礼

{宛先の名前} 様

お疲れ様です。{自分の名前}です。

先日は{内容}にご対応いただき、誠にありがとうございました。
{具体的に助かった点を1-2文}

今後ともよろしくお願いいたします。

{自分の名前}
```

---

## テンプレート4: 謝罪メール

```
件名: 【お詫び】{件名}について

{宛先の名前} 様

お疲れ様です。{自分の名前}です。

{事象}につきまして、ご迷惑をおかけし誠に申し訳ございません。

■ 事象
- {何が起きたか}

■ 原因
- {原因}

■ 対応状況
- {実施済みの対応}

■ 再発防止策
- {防止策}

今後このようなことがないよう十分注意してまいります。
重ねてお詫び申し上げます。

{自分の名前}
```

---

## テンプレート5: 会議案内メール

```
件名: 【会議案内】{会議名}（{日付}）

各位

{自分の名前}です。
以下の通り会議を開催いたします。ご出席のほどよろしくお願いいたします。

■ 会議名: {会議名}
■ 日時: {YYYY年MM月DD日 HH:MM - HH:MM}
■ 場所: {場所 / オンライン URL}
■ 参加者: {参加者一覧}

■ アジェンダ
1. {議題1}（{担当者}、{所要時間}）
2. {議題2}（{担当者}、{所要時間}）
3. その他・Q&A

■ 事前準備
- {準備事項}

ご都合が悪い場合は事前にご連絡ください。

{自分の名前}
```

---

## Claude Code 活用のコツ

- **トーン調整**: 「もっとカジュアルに」「よりフォーマルに」と追加指示で調整可能
- **翻訳**: 「英語版も作成して」で多言語対応
- **添削**: 自分で書いた下書きを渡して「ビジネスメールとして添削して」も有効
- **一括生成**: 「同じ内容で宛先3名分を作成して」で複数メールを一括作成
