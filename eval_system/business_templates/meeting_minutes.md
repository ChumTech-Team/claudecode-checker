# 議事録テンプレート

> Claude Code で議事録を作成・整理する際に使用するテンプレート

---

## 使い方

Claude Code に以下のようにプロンプトを投げる:

```
以下の文字起こしデータから議事録を作成してください。
テンプレートは eval_system/business_templates/meeting_minutes.md を参照。
```

または会議メモを渡して整理を依頼:

```
以下の会議メモを議事録形式に整理してください。
[メモを貼り付け]
```

---

## 標準議事録テンプレート

```markdown
# 議事録: {会議名}

| 項目 | 内容 |
|------|------|
| 日時 | YYYY年MM月DD日 HH:MM - HH:MM |
| 場所 | {場所 / オンライン} |
| 参加者 | {名前1}, {名前2}, {名前3} |
| 欠席者 | {名前} |
| 記録者 | {名前} |

---

## アジェンダ

1. {議題1}
2. {議題2}
3. その他

---

## 議事内容

### 1. {議題1}

**報告・説明:**
- {要点1}
- {要点2}

**議論:**
- {発言者A}: {発言内容}
- {発言者B}: {発言内容}

**決定事項:**
- {決定1}

---

### 2. {議題2}

**報告・説明:**
- {要点}

**議論:**
- {発言内容}

**決定事項:**
- {決定}

---

## アクションアイテム

| # | タスク | 担当者 | 期限 | ステータス |
|---|--------|--------|------|----------|
| 1 | {タスク内容} | {担当} | MM/DD | 未着手 |
| 2 | {タスク内容} | {担当} | MM/DD | 未着手 |
| 3 | {タスク内容} | {担当} | MM/DD | 未着手 |

---

## 次回会議

- 日時: YYYY年MM月DD日 HH:MM
- 議題（予定）: {次回議題}

---

## 備考

- {補足事項}
```

## 簡易議事録テンプレート（定例会議用）

```markdown
# {会議名} 議事録 - YYYY/MM/DD

参加者: {名前1}, {名前2}

## 報告事項
- {報告1}
- {報告2}

## 議論・決定事項
- {決定1}
- {決定2}

## アクションアイテム
- [ ] {タスク1}（{担当}, {期限}）
- [ ] {タスク2}（{担当}, {期限}）

## 次回: MM/DD
```

## Claude Code 活用フロー

### 文字起こしからの議事録作成

```
1. 録音データ → 文字起こし（meeting_analysis）
2. 文字起こしテキストを Claude Code に投入
3. 「上記テンプレートで議事録を作成して」と指示
4. 議事録を確認・修正
5. notion スキルで共有・タスク登録
```

### 議事録からのアクションアイテム抽出

```
この議事録からアクションアイテムを抽出して、
担当者・期限付きの一覧表にしてください。
```

### 議事録の要約

```
この議事録を3行で要約してください。
経営層向けの報告に使います。
```
