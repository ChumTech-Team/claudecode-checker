## 全体マップ

---

## 設計方針

- **対象者**: 社長業（開発・リサーチ・資料作成・クライアント対応・チームマネジメント）
- **ゴール**: Claude Codeの全機能を理解し、全業務を効率化できるプロフェッショナルになる
- **将来用途**: 各ナレッジファイルが評価エージェントのナレッジベースに直結（1ファイル = 1評価ドメイン）
- **制約**: 各ファイル150ルール以内（遵守率95%維持ライン）
- **品質保証**: 全ファイルが research/ の元データからのみ作成（hallucination防止）

### Week 1-2: 初級
- **読む**: 01_基礎と設定、03_コア機能

- **実践**: CLAUDE.md作成、基本ファイル操作、Git連携

### Week 3-4: 中級
- **読む**: 02_コンテキストエンジニアリング、04_拡張機能
- **Academy**: MCP入門、Agent Skills入門
- **実践**: MCP 3-5個導入、カスタムスキル作成、Strategic Compact実践

### Week 5-6: 上級
- **読む**: 05_自動化と並列実行、06_ワークフローと方法論
- **Academy**: MCP上級、Claude Code in Action（上級/エキスパート）
- **実践**: Hooks設定、worktree並列、Plan-Firstワークフロー

### Week 7-8: エキスパート
- **読む**: 07_業務別活用、08_セキュリティ、09_コストと運用
- **Academy**: Building with Claude API
- **実践**: GitHub Actions導入、セキュリティ監査、ROI測定
- **認定**: Claude Certified Architect (CCA) Foundations 受験準備
